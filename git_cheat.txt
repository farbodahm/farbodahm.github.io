git init - Initialize a Git repo

git status - Give status of the Repo

git add FILES - Tells git to track those new files

git reset FILES - Tells git not to track those files

git log --summery - To Give logs with full details 

git remote add origin https://github.com/username/repo - Add url as a remote

git push origin master - Pushing all local commits to remote origin branch master

git pull orogin master - Check for changes and pull down

git diff HEAD - Show changes from our last commit

git checkout -- FILE.TXT - Get rid of all the changes from last commit to FILE.TXT till now


## Branchs:

git branch - Show current branchs

git branch BRANCHNAME - Create branch 

git branch -m BRANCHNAME NEWBRANCHNAME - Rename the branch

git branch -D BRANCHNAME - Delete the branch

git checkout BRANCHNAME - Switch to the branch

git merge BRANCHNAME - Merge the branch to current branch
